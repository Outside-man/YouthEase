# 大学生“易” YouthEase
小组成员：沈梦芸、殷晓明、王宇枫

面向大学生为主题的交易网站，可以二手交易，转让转租，甚至可以交易声音等特色功能。
UI参照百度贴吧
实现功能：
1.【用户方】用户注册，登录，添加修改个人信息，发布、删除、修改交易贴、水帖、感情交流贴，添加好友，发动交易等功能；
2.【管理员方】后台登录，修改、删除帖子信息、用户信息，拉用户黑名单等功能；
3.【网站功能】交易帖、感情贴、杂帖等三大块，支持上传文字、图片、视频，收藏及点赞帖子，搜索帖子等功能。
